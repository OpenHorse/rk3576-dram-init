# RK3568 raminit RFC (architectural template)

This project mirrors the structure of Pavel Golikov's RK3568 BLOB-free TPL
DRAM init RFC. That series is the primary architectural template for the
RK3576 work.

- Series: `[PATCH RFC 00/18] rockchip: rk3568: add DRAM controller driver and
  initialize DRAM at TPL stage`
- Author: Pavel Golikov `<paullo612@ya.ru>`
- U-Boot list thread:
  https://lore.kernel.org/u-boot/?q=rk3568+raminit

Refer to the upstream thread rather than a local copy so the reference stays
current with review feedback.
